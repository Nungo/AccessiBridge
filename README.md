# AccessiBridge 🌉

**AI-Powered Accessibility Companion for the Visually and Hearing Impaired**

AccessiBridge bridges the gap between technology and accessibility, empowering 3.7 billion people worldwide with AI-driven visual and auditory assistance.

## 🎯 The Problem

- **2.2 billion people** have visual impairments globally
- **1.5 billion people** experience hearing loss
- Current assistive technologies are expensive, fragmented, and require multiple apps
- Real-time environmental awareness remains a critical unmet need

## 💡 The Solution

AccessiBridge is a unified web application that provides:

### 👁️ **Visual Assistance** (for Blind/Low Vision Users)
- **Scene Description**: AI-powered detailed environmental analysis
- **Text Reading**: OCR with text-to-speech for documents, signs, menus
- **Navigation Safety**: Real-time obstacle detection and path guidance
- **Voice Control**: Hands-free operation with natural language follow-up questions
- **Conversation Memory**: Context-aware responses across sessions

### 👂 **Hearing Assistance** (for Deaf/Hard of Hearing Users)
- **Live Captions**: Real-time speech-to-text transcription
- **Sound Detection**: AI-powered environmental sound classification
- **Visual Alerts**: Screen flashes for important sounds
- **Haptic Feedback**: Vibration alerts for urgent sounds (fire alarm, doorbell)
- **Export Logs**: Save transcripts and sound events for later review

## 🛠️ Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **AI**: Google Gemini 2.5 Flash (vision + text analysis)
- **Speech**: Web Speech API (captions + voice control)
- **Audio**: Web Audio API (volume-based sound detection)
- **Hosting**: Vercel

## ✨ Key Features

- **Fully Responsive**: Works on desktop, tablet, and mobile
- **Real-Time Processing**: Instant AI analysis and feedback
- **Offline-Ready Audio**: Browser-based speech recognition (no API calls)
- **Accessible UI**: High-contrast glass morphism design
- **Privacy-First**: All processing happens client-side or via secure APIs

## 🏗️ Local Development
```bash
# Clone repository
git clone https://github.com/Nungo/AccessiBridge.git
cd AccessiBridge

# Install dependencies
npm install

# Add environment variables
# Create .env file with:
# VITE_GEMINI_API_KEY=your_gemini_api_key

# Run development server
npm run dev
```
